This test utilizes the Navigation Timing API to give you a ballpark estimate of how long a certain asset or common site configuration takes to load.

The result is arrived at by calculating performance.loadEventEnd - performance.requestStart. This determines the time (in ms) from the browser starts requesting resources from the server (after DNS has been resolved) to the moment the load event finishes.

Each test is run ten times. On completion, the arithmetic mean is displayed.
